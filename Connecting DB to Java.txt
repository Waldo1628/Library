Connecting DB to Java Program:

First Steps (Downloading SQL Connector):
=> Create a libs directory inside your project directory
=> Place the sql connector jar file inside the libs folder created


Connecting SQL connector to IDE:
=> File => Project Structures => Modules from the left pane => Make sure your project name is selected
=> Select "Dependencies" => Click on the '+' icon and select "JARs or Directories" => Select Apply and then OK
